# Formatting tips

* **.rst:** You can  write your documentation pages in rst format (close to markdown). See here for some references on the rst syntax: [https://thomas-cokelaer.info/tutorials/sphinx/rest_syntax.html](https://thomas-cokelaer.info/tutorials/sphinx/rest_syntax.html)


* **.md:** You can also use markdown format if you have switched on the `myst-parser` option in the `conf.py` file. To do this you should add the extension `myst_parser` and the `source_suffix` list. You should also add  `myst_enable_extensions = ["dollarmath", "amsmath"]` to improve  the associated latex integration.
See [https://myst-parser.readthedocs.io/en/latest/syntax/optional.html](https://myst-parser.readthedocs.io/en/latest/syntax/optional.html) for more info.


* **latex integration:**

Then you can write some latex:

$$
   \begin{eqnarray}
      y    & = & ax^2 + bx + c \\
      f(x) & = & x^2 + 2xy + y^2
   \end{eqnarray}
$$
